# Archive Directory

This directory contains archived files that are no longer actively used but are kept for reference.

## Structure

- **setup/**: Git setup scripts and documentation
- **migrations/**: Migration guides and completion reports
- **integrations/**: Integration implementation summaries
- **backups/**: Backup archive files (.zip)

## Note

These files are archived for historical reference. For current documentation, see the root `README.md` or `docs/` directory.
